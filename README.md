# FiveChess
一个java控制台小游戏（五子棋）
前言： 该项目 为 Java 版五子棋小游戏，为控制台小游戏，主要功能有 人机大战,左右互战，可以悔棋可以提示，

人机大战采用 五元贡献法，代码简单 实况效果好 ，相比 一般 博弈论 判断局面的方式，比正真的AI 还是有一定的差距。
